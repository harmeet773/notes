Process

An instance of a program with its own address space, resources, and execution state.

Thread

A smaller execution unit within a process that shares the same memory with other threads inside the same process .

Process

process - A process is an independent program in execution.

Key Points
  • Has its own memory (RAM space).
  • Has its own CPU time, stack, heap, file handles.
  • Crashing one process doesn't affect others.
  • Examples: Chrome.exe, VSCode.exe, python.exe

Thread

What is a Thread? A thread is the smallest unit of execution inside a process.

Key Points
  • Threads share the memory of the process.
  • Lightweight compared to a process.
  • One process can have multiple threads.
  • Examples: A browser tab, file download thread, render thread.
