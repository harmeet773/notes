JavaScript & React Basics

  • Difference between arrow function and normal function?
    • Arrow functions don’t have their own this or arguments object.
    • this is captured lexically from the surrounding scope.
  • Closure — A closure is created when a function remembers and can access variables from its outer (lexical) scope, even after the outer function has finished executing.
  • Async, await, promise — A Promise has 3 states: pending, fulfilled, rejected. async/await provides a cleaner way to work with promises.
  • React component extends which class? — Class components extend React.Component.
  • Can functional components implement all features without hooks? — Hooks are required to use state, lifecycle, context, etc., in functional components.

Forms

  • Controlled component — React state controls the form input value.
  • Uncontrolled component — The DOM manages the value, typically accessed via refs.

State Management & Optimization

  • Prop drilling — Passing props down multiple levels unnecessarily; avoid with Context API or state management libraries like Redux.
  • useMemo and useCallback
    • useMemo memoizes expensive computed values.
    • useCallback memoizes function references to prevent unnecessary re-renders of child components.

Lists & Keys

  • When to use keys? What happens if we do not use a key?
    • Use stable keys for list items to help React track changes efficiently.
    • Without keys (or with unstable keys), React may re-render or reorder items incorrectly, causing visual glitches and performance issues.

HTML Tags

  • <link> vs <a>
    • <link> is used in the document head to include external resources (e.g., CSS, preloads).
    • <a> creates a hyperlink to navigate to another URL or section.

Utilities

  • Debounce function — example implementation 
    
// Simple debounce implementation
function debounce(fn, delay) {
  let timerId;
  return function (...args) {
    const context = this;
    clearTimeout(timerId);
    timerId = setTimeout(() => fn.apply(context, args), delay);
  };
}
