Techi Tech

  • Supabase is an open-source backend platform built around PostgreSQL.
  • It provides database, authentication, storage, real-time subscriptions, and serverless edge functions.
  • It is commonly used as a backend for web apps, mobile apps, prototypes, and AI-generated applications.

  • Database: Managed PostgreSQL with tables, SQL editor, relationships, and policies.
  • Auth: User sign-up, login, OAuth providers, magic links, and session handling.
  • Storage: File buckets for images, documents, and user-uploaded assets.
  • Realtime: Listen to database changes and update the UI instantly.
  • Edge Functions: Run secure backend logic close to users.

  • Adding login and user profiles to an application.
  • Saving form submissions, dashboard data, or app records in PostgreSQL.
  • Building admin panels, internal tools, SaaS apps, and MVPs.
  • Connecting generated apps from tools like Lovable to a real backend.

  • Always configure Row Level Security policies before exposing tables to users.
  • Never place service role keys in frontend code.
  • Use migrations or SQL scripts to track important schema changes.
  • Test authentication, permissions, and file access before production use.

  • Slackbot is Slack's built-in assistant for reminders, automated responses, and simple workspace help.
  • It can send direct messages, remind users about tasks, and answer custom trigger phrases configured by workspace admins.
  • It is useful for lightweight automation that does not need a full Slack app or workflow.

  • Creating personal reminders with commands like /remind.
  • Sharing quick answers for repeated workspace questions.
  • Pointing teammates to common links, policies, forms, or support channels.
  • Sending helpful onboarding notes when someone uses a specific keyword or phrase.

  • Keep Slackbot responses short so they are easy to scan inside busy channels.
  • Use clear trigger phrases to avoid accidental or noisy replies.
  • Review custom responses regularly so links, policies, and team details stay current.
  • Use Slack workflows or Slack apps when the automation needs approvals, integrations, or complex logic.
