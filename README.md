"# notes" 


this repository contains notes.

live at https://harmeet773.github.io/notes/

## Building

Run the build script before deploying to regenerate the search index:

```bash
./build.sh
```

This will:
- Scan all HTML pages in the root directory
- Extract titles and content
- Generate `search.json` with `/notes/` baseurl paths

## Search setup

- The search index is stored in `search.json`
- It's automatically generated by the build script
- Serve the site over HTTP for search to work:

```bash
python3 -m http.server 8000
```

Then visit: http://localhost:8000/notes/
